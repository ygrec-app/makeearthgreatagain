# MEGA — website

Astro 6 + Svelte 5 + Threlte 8 rewrite of the MEGA portfolio. Lenis smooth scroll, GSAP motion, Threlte megaverses. Design tokens emit from `../design.md`.

## Dev

```bash
npm install
npm run dev      # http://localhost:4321
npm run build
npm run build-tokens   # regenerate src/styles/tokens.css from ../design.md
```
